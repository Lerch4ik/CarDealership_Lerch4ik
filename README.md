# CarDealership

Демонстрационное desktop-приложение для управления автосалоном, созданное в учебных целях.
Несмотря на это, проект имеет полноценную архитектуру и может служить основой для реального внедрения.

## О проекте

Приложение автоматизирует основные бизнес-процессы автосалона: ведение каталога автомобилей,
учёт клиентов и менеджеров, оформление продаж и сервисного обслуживания.

## Возможности

- 🚗 **Автомобили** — каталог с характеристиками (марка, модель, год, VIN, тип топлива, КПП, кузов, статус)
- 👤 **Клиенты** — база клиентов с паспортными данными, контактами и адресом
- 👔 **Менеджеры** — управление сотрудниками, должности, статус активности
- 💰 **Продажи** — оформление сделок, скидки, способы оплаты, привязка к менеджеру
- 🔧 **Сервис** — записи о техническом обслуживании, статус, стоимость, техник
- 📊 **Дашборд** — сводная статистика по ключевым показателям
- 🌗 **Темы** — светлая и тёмная тема оформления

## Стек технологий

- C# / WPF (.NET)
- Entity Framework Core
- SQL Server
